# reverie-batteries

Batteries included for reverie/CMS.

Following included:

- objects
  - reverie/image (basic image object)
  - reverie/raw (will put out the raw output exactly as is, handle with care)
  - reverie/text (basic text object)

## Usage

```clojure
[reverie-batteries "0.3.2"]
```
